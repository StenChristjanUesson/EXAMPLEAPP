1. Ava terminal kaustas kus on docker-compose.yml (ITB2203Application kaust)
2. Dockeri programm peab arvutis hetkel käima
3. Käivita api ja postmani testid käsuga: docker-compose up --build 
4. Terminali tulevad testide tulemused
5. Uuesti käivitamiseks saab hektel terminalis käiva protsessi peatada hoides CTRL ja C nuppe
6. Uuesti käivitamiseks tuleb konteiner eemaldada ja uuesti buildida, käsk eemaldamiseks: docker-compose down
7. Uuesti 3. sammust

NB: Terminali ei pea kinni panema.
